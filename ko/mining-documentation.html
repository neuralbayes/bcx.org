다큐멘테이션

BitcoinX마이닝 입문

1. 월렛주소개설

전노드월렛： Github다운

BitcoinX Core월렛, Bitgo월렛혹은 Bitpie사용으로 당신의BCX주소 생성

2.마이닝 프로그램 다운

Windows
다운로드 gominer
다운로드 run.bat
Linux
다운로드 gominer-v1.0-linux
다운로드 run.sh

3. 마이닝 프로그램 셋업

Windows

gominer-v1.0-windows.exe과 run.bat 동시 파일에 저장, 오른쪽 마우스 클릭“run.bat”, “편집”클릭, 마이닝풀 선택에 따라, 마이닝풀 주소와 월렛주소를 입력, run.bat 더블클릭.

Linux

gominer-v1.0-linux과 run.sh 동시 파일에 저장, run.sh편집, 당신이 선택한 마이닝풀의 설명에따라, 마이닝풀 주소와 월렛주소를 입력, run.bat 더블클릭.

4.BCX마이닝풀

4.1 QuickPool (신청수요)

오른쪽 위부분의 '신청'을 클릭후, 이용자이름, 메일과 비밀번호 입력
--url=stratum+tcp:/.quickpool.com:3334 --user=your_bcx_address.workid

QuickPool방문

4.2 VVPool (무신청수요)

gominer --url=stratum+tcp:/.vvpool.com:5660 --user=you_bcx_address.workid

VVpool방문

4.3 liu.so (신청수요)

gominer-v1.0-windows.exe -url stratum+tcp://liu.so:5766 -user=username.workid -I 28

liu방문
