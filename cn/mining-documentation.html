挖矿文档

BitcoinX挖矿入门

1. 创建钱包地址

全节点钱包： 从github下载

或者使用BitcoinX Core钱包，Bitgo钱包或Bitpie注册创建您的BCX地址。

2.下载挖矿软件

Windows
Download gominer
Download run.bat
Linux
Download gominer-v1.0-linux
Download run.sh

3.设置挖矿软件

Windows

将gominer-v1.0-windows.exe和run.bat放在同一目录下，右键单击“run.bat”，然后单击“编辑”，按照您选择的矿池的说明，输入矿池地址和钱包地址。然后双击run.bat。

Linux

将gominer-v1.0-linux和run.sh放在同一目录下，编辑run.sh，按照您选择的矿池的说明，输入矿池地址和钱包地址。然后双击run.sh或在命令行中输入sh run.sh。

4.BCX矿池

4.1 QuickPool (需要注册)

单击右上角的“注册”，然后输入用户名，电子邮件和密码。
--url=stratum+tcp:/.quickpool.com:3334 --user=your_bcx_address.workid

访问QuickPool

4.2 VVPool (无需注册)

gominer --url=stratum+tcp:/.vvpool.com:5660 --user=you_bcx_address.workid

访问VVpool

4.3 liu.so (需要注册)

gominer-v1.0-windows.exe -url stratum+tcp://liu.so:5766 -user=username.workid -I 28

访问liu
